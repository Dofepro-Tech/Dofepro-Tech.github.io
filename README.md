# Biblia NJ

Aplicación de lectura, exploración y estudio bíblico con cliente React, backend Express, soporte de IA vía servidor, búsqueda global en la Biblia y empaquetado móvil con Capacitor para Android.

## Qué Incluye

- Lectura por libro, capítulo y versículo.
- Búsqueda global de palabras o frases en la Biblia.
- Marcadores, resaltados y preferencias guardados localmente.
- Lectura en voz alta de capítulos y versículos.
- Estudio guiado, explicación y chat bíblico con IA vía backend.
- Retos diarios, racha, recompensas y contenido devocional diario.
- Interfaz bilingüe en español e inglés.
- Build web y Android desde la misma base de código.
- Inicio rediseñado con branding propio, splash temático y panel interno listo para integrar futuros juegos bíblicos.

## Uso Local

```bash
npm install
copy .env.example .env.local
npm run dev
```

Entorno local por defecto:

- Frontend: [http://localhost:3000](http://localhost:3000)
- Backend: [http://localhost:3001](http://localhost:3001)

Vista previa local:

```bash
npm run preview
```

Esto levanta la web en [http://localhost:3000](http://localhost:3000) y el backend en [http://localhost:3001](http://localhost:3001).
